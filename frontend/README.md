# Cyber Threat Intelligence Dashboard

A comprehensive, full-stack Cyber Threat Intelligence application designed to visualize, analyze, and report on global cyber threats in real-time. This system combines a high-performance React frontend with a robust Python/Flask backend and integrates advanced AI models for automated threat analysis.

## üöÄ Key Features

*   **Real-time Threat Map**: Interactive 3D globe visualizing live cyber attacks and threat origins using D3.js and React Simple Maps.
*   **Live Intelligence Feed**: Real-time news aggregation via Server-Sent Events (SSE), featuring flip-card animations for latest cybersecurity headlines.
*   **AI-Powered Briefings**: Automated "Daily Threat Briefings" generated by Large Language Models (Groq / Llama 3), summarizing complex threat data into actionable insights.
*   **Malicious IP Analysis**: On-demand risk assessment of suspicious IP addresses using AbuseIPDB and AI simulation for unparalleled context.
*   **Automated Reporting**: One-click generation of professional PDF reports including charts, attack statistics, and AI summaries.
*   **Advanced Analytics**: Visualizations of attack vectors, top targeted nations, and historical trends.

## ‚ö° Installation & Setup

### 1. Backend Setup

The backend handles data aggregation and AI processing.

```bash
# Navigate to the backend directory
cd cyber_threat_intel

# Create a virtual environment (optional but recommended)
python -m venv venv
# Windows
venv\Scripts\activate
# Unix/MacOS
source venv/bin/activate

# Install dependencies
pip install flask flask-cors python-dotenv aiohttp groq
```

### 2. Frontend Setup

The frontend delivers the interactive dashboard.

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

## üñ•Ô∏è Usage

1.  Start the backend server:
    ```bash
    python server.py
    ```
    The server will start on `http://localhost:5000`.

2.  Start the frontend:
    ```bash
    npm run dev
    ```
    Access the dashboard at the URL provided by Vite (typically `http://localhost:5173`).

3.  **Explore**:
    *   Navigate to the **Dashboard** for the live map and stats.
    *   Check the **News Feed** for real-time updates.
    *   Use the **Briefing** feature to get an AI-generated summary of the current threat landscape.
    *   Generate a **PDF Report** for offline analysis.
